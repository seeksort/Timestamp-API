Timestamp Microservice API

Insert either a natural language date or Unix date after the / in the address, and this site will provide a JSON object with both formats.

Example Input:

https://timestamp-ms.herokuapp.com/Oct%2024,%202016
https://timestamp-ms.herokuapp.com/1477285200

Example Output:

{ "unix": 1477285200, "natural": "October 24, 2016" }
